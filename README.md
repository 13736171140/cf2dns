# cf2dns
cf2dns宝塔插件版

#### 介绍
本项目为cf2dns宝塔插件版。    
主要用于将CloudFlare优选IP地址解析到您的域名记录中。
后续降逐渐支持CloudFront Gcore等其他CDN服务。

#### 使用说明  

将本项目ZIP包下载到电脑，打开面板-软件商店-第三方应用-导入安装即可
